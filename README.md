# Тестирование API для сервиса Stellar Burger

В этом проекте реализованы автоматизированные тесты для функций API сервиса Stellar Burger

## Проверенные функции

- **Создание уникального пользователя** - проверка возможности создания уникального пользователя, включая негативные сценарии  
- **Логин пользователя** - проверка авторизации пользователя по email и password, включая негативные сценарии
- **Создание заказа** – проверка возможности создания нового заказа, включая различные параметры и негативные сценарии  

Инструкция по запуску:

1. Установите зависимости:
pip install -r requirements.txt

2. Запустить все тесты и записать отчет:
pytest tests --alluredir=allure_results

3. Посмотреть отчет по прогону html
allure serve allure_results
